# Sistema de Turnos Médicos 
Trabajo Final 
Materia Programacion 2 

## Tecnologías
Node.js, Express, MongoDB, JWT

## Instalación
npm install
npm run dev

## Variables de entorno
MONGODB_URI=
JWT_SECRET=
PORT=

## Funcionalidades

- Registro/Login
- Roles (admin/cliente)
- CRUD turnos
- Gestión de profesionales

## Endpoints principales

POST /api/auth/register
POST /api/auth/login
GET /api/turnos

## Seguridad

- JWT Authentication
- Password hashing con bcrypt
- Middleware de roles
